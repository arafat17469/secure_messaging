Badri 313 Database Security View

This page is for project demonstration. It proves that user passwords are stored as bcrypt hashes, messages are stored as encrypted ciphertext, and RSA private keys are not stored in the database.

Back to Main App

Users Table

Plaintext password is not stored. bcrypt password hash is stored.

RSA private key is not stored in the database. Only RSA public key is stored.

username_display is kept for login and receiver selection. username_hash is stored to demonstrate identity hashing.

Messages Table

Plaintext messages are not stored. Only encrypted ciphertext, nonce, AES tag, SHA-512 hash, RSA signature, and UUID are stored.
